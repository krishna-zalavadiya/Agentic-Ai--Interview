# AI Interview Platform

An AI-powered mock interview platform that helps candidates practice technical and HR interviews with real-time feedback.

## Features
- AI-based interview questions
- Resume-based interview generation
- Voice interview simulation
- Performance analytics dashboard
- Confidence and communication scoring
- PDF feedback report

## Tech Stack
Frontend: React.js, TailwindCSS
Backend: Node.js, Express.js
Database: MongoDB
AI: OpenAi/LLM integration
